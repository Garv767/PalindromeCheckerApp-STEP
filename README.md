# PalindromeChecker App

The objective of the PalindromeChecker App is to design and implement a console-based Java application that validates whether a given string is a palindrome under different conditions. This project serves as a comprehensive learning path for strengthening core programming fundamentals, applying design patterns, and understanding data structure concepts.


## 📂 Project Structure

The project is divided into use cases, each contained in its own branch for clarity and progression:

```
└── 📁PalindromeCheckerApp-STEP
    └── PalindromeCheckerApp.java
```

## 🛠️ Key Concepts Covered

*   **Java Fundamentals**: Loops, conditionals, and String manipulation.
*   **Data Structures**: Utilization of `Stack`, `Deque`, and `ArrayDeque`.
*   **Design Patterns**: Implementation of the **Strategy Pattern** for extensible algorithm design.
*   **Object-Oriented Programming (OOP)**: Interfaces, Polymorphism, and Encapsulation.

## ⚙️ Installation & Usage

To run any specific use case, follow these steps in your terminal:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Garv767/PalindromeCheckerApp-STEP
   cd PalindromeCheckerApp-Step

2. **Compile a specific Use Case (e.g., UC13):**
    ```bash
    git checkout feature/UC13
    javac PalindromeCheckerApp.java

3. **Run the application:**
    ```bash
    java PalindromeCheckerApp
